# JKNet-dgl

A DGL([Deep Graph Liabrary](https://www.dgl.ai/)) and [PyTorch](https://pytorch.org/) implementation of Jumping Knowledge Networks ([https://arxiv.org/abs/1806.03536](https://arxiv.org/abs/1806.03536)).

## Prerequisites

- Python >= 3.7
- numpy >= 1.15.4
- scikit-learn >= 0.20.0
- PyTorch >= 1.0.0
- dgl >= 0.1.3
